Dribdat Data Package
====================

This is an export of data about events, projects, participants and activities
generated with the dribdat application for hackathons and other collaborative,
time-bounded events.

You can use this file to generate a static site using the backboard tool,
available at: https://github.com/dribdat/backboard#usage

The Data Package standard is explained at: https://frictionlessdata.io/standards

For more information and support on dribdat, please visit https://dribdat.cc
